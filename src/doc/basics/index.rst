OpenLayers Basics
=================

`OpenLayers 3 <http://ol3js.org>`_ is a library for building mapping applications in a browser. This workshop covers the new 3.x series. The library lets developers integrate data from a variety of sources, provides a friendly API, and results in engaging and responsive mapping applications.

This module introduces fundamental OpenLayers concepts for creating a map.

.. rubric:: What this module covers

In this module you will create a map, dissect your map to understand the parts, and get links to additional learning resources.
